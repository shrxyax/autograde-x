# AutoGradeX

AI-Assisted Course Project Evaluation System

## Features

- Student assignment submission
- AI-based grading
- AI detection probability
- Faculty grading override
- Role-based access control
- Admin user management

## Tech Stack

- Next.js
- TypeScript
- Prisma
- SQLite
- Groq LLM API
- TailwindCSS

## System Roles

Student  
Faculty  
Admin  

## Setup

Install dependencies:
npm install
Run database migration:
npx prisma migrate dev

Start server:
npm run dev

## Environment Variables.

DATABASE_URL
JWT_SECRET
GROQ_API_KEY
